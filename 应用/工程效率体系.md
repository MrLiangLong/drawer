### 工程化价值和定位
1. 标准化前端工作流，统一团队开发方式     
2. 团队技术沉淀、新技术探索和实施利器    
3. 减少重复造轮子，提升轮子质量  

### 工程考量三要素
1. 质量  
2. 性能  
3. 效率  

### 工程典型周期  
1. 创建项目
    创建git仓库
    脚手架生成代码
    创建某个页面
2. 本地开发
    打包构建
    接口联调
    组件平台
    数据配置平台
3. 功能测试
    移动测试代理
    单元测试
    端到端测试
4. 发布
    代码规范检查
    Code Review
    CDN发布
    Node服务发布
    离线包发布
5. 线上监控
    JS错误监控
    接口成功率
    页面访问波动
    测试监控
    质量日报
